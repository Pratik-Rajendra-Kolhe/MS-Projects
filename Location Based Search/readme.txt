1. After Dump processing and after setting up solr on apache tomcat on port 8080 and indexing the dump, we have one instance of apache tomcat running solr.
2. Create a new Java EE project (with project name Search) in Eclipse and import/copy-paste processing code from folder 'processor' in the zip submitted.
3. Make sure that the folder structure is the same in the newly created project as that in the zip.
4. Select apache tomcat as the default server to run the Java EE application on port 8888.
5. Make sure to have google chrome browser installed in C:\Program Files (x86)\Google\Chrome\Application\chrome.exe
6. Run the project on apache tomcat server.
7. Open your browser and goto localhost:8888/Search to perform location based news search. :)
